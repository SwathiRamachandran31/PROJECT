# PROJECT

Company name : KADA Die Lebensmittel

Description: This supermarket is based out of Germany offering Indian, Persian, and German groceries, concentrating on Indian spices, fruits, and vegetables imported directly from India.This project will be based on information system focusing product inventory management.By this project the staff in the supermarket can improve their product management like adding new product, viewing current stock, updating details like price and quantity of certain products and deleting it when they are no longer sold or expired.
